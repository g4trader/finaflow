## 🛠️ Comandos Principais

```bash
# Iniciar tudo
docker-compose up --build

# Ver logs em tempo real
docker-compose logs -f

# Conectar ao banco
psql -h localhost -U finaflow_user -d finaflow

# Parar (preserva dados)
docker-compose stop

# Parar e limpar (inclui volume)
docker-compose down -v

# Executar comando no backend
docker-compose exec backend bash

# Ver status
docker-compose ps
```

## 💡 Diferenças Local vs Cloud

| Aspecto | Cloud (Antes) | Local (Agora) |
|---------|---------------|--------------|
| Startup | 30-60s | 5-10s |
| Hot-reload | Não | ✅ Sim |
| Latência | 500-2000ms | <100ms |
| Debugging | Remote logs | Local stdout |
| Onboarding seed | 10+ min → timeout | Segundos |
| Cost | Servidor 24/7 | Zero |
